# ДЗ 10 — PoC (эмуляция) для CVE

## CVE
**CVE-2021-41773**

## Краткое описание
CVE-2021-41773 — уязвимость Path Traversal (обход пути) в Apache HTTP Server 2.4.49.
При определённых условиях атакующий может сформировать специальный URL-путь, который позволяет
обращаться к файлам вне корневой директории веб-сервера (например, попытаться прочитать `/etc/passwd`).

## Потенциальный ущерб
- Чтение произвольных файлов (утечки конфигураций, ключей, паролей)
- Подготовка к дальнейшей компрометации

## Что делает PoC в этом репозитории
Скрипт `poc.py` реализует **эмуляцию взаимодействия**:
- формирует payload (подозрительный путь запроса) для Path Traversal
- формирует пример "сырого" HTTP-запроса (строкой)
- выводит сообщение о потенциальной атаке
