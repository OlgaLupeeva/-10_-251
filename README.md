# ДЗ 10 — PoC для CVE (эмуляция)

## CVE
**CVE-2021-41773**

## Краткое описание
CVE-2021-41773 — уязвимость Path Traversal (обход пути) в Apache HTTP Server 2.4.49.
Атакующий может сформировать специальный URL, который позволяет обратиться к файлам вне
корневой директории веб-сервера (например, попытаться прочитать `/etc/passwd`).

## Потенциальный ущерб
- Чтение произвольных файлов (утечки конфигураций/ключей/паролей)
- Подготовка к дальнейшей компрометации

## Что делает PoC
Скрипт формирует запрос к условно уязвимой точке и выводит сообщение о потенциальной атаке.
По умолчанию работает в режиме dry-run (запрос не отправляется). При флаге `--send` отправляет
HTTP GET запрос на указанный URL (использовать только на тестовом окружении).
